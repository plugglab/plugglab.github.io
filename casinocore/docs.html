Economy

Player Economy Placeholders

Economy placeholders provide access to balances and hooked economy provider information.

%casino_balance%

Formatted player balance.

Example Output $12,500
%casino_balance_raw%

Raw player balance with 2 decimal places.

Example Output 12500.42
%casino_economy%

Current hooked economy provider name.

Example Output Vault
