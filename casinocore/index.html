Casino Core

Slot machines, roulette tables, and a configurable reward shop — a full casino economy layer that plugs into Vault.

Download on Modrinth Discord Support

Capabilities

What Casino Core does

Casino Core gives economy servers a ready-made gambling layer without hand-rolling GUI menus or payout math.

Slot Machines

Configurable reel odds, jackpot pools, and animated GUI slot machines players can place on their own claims.

Roulette Tables

Full roulette with configurable bet types, payout multipliers, and a shared table state for multiple players.

Vault Economy Sync

Every wager and payout runs through Vault, so Casino Core works with whatever economy plugin your server already uses.

Reward Shop

A configurable token shop lets players redeem casino winnings for cosmetics, kits, or custom commands.

Admin Controls

Set table limits, house edge, and daily wager caps per-world or per-permission group.

PlaceholderAPI Support

Show live balances, win streaks, and jackpot totals in scoreboards, tab lists, or chat via placeholders.
