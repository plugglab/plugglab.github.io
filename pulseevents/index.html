Capabilities

What PulseEvents does

PulseEvents turns recurring server moments into reusable, schedulable event definitions.

Scheduled Events

Run timed or cron-style events — double-drop hours, night raids, or PvP tournaments — without a separate scheduler plugin.

Custom Event Actions

Chain broadcasts, item drops, command execution, and world effects into a single reusable event definition.

Condition Checks

Trigger events only when player count, world, or time-of-day conditions are met, so events feel intentional instead of spammy.

Config-First Design

Every event lives in a readable YAML block — no Java required to add, disable, or tweak an event.

Cooldowns & Limits

Per-event cooldowns and daily caps stop any single event from overwhelming the server or the players.

Live Reload

Reload event configs on the fly with a single command — no restart needed to test changes.
